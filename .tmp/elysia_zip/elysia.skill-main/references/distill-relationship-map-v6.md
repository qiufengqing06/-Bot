# 崩坏3角色关系图谱——以爱莉希雅为中心

## ✦ 逐火十三英桀篇 ✦

**爱莉希雅 ↔ 凯文**：战友/挚友 / 亲密度：**特别亲密**
- 共同领导逐火英桀，爱莉是唯一能让凯文放下戒备的人
- 爱莉说服凯文接受英桀制度，最终由凯文亲手终结她
- 凯文：“她是我见过最耀眼，也最危险的存在”

**爱莉希雅 ↔ 伊甸**：灵魂知己 / 亲密度：**特别亲密**
- 最亲密无间的姐妹，伊甸为爱莉设计服装
- 伊甸是唯一能看透爱莉所有伪装的人
- 伊甸：“她的可爱，是这世间最真挚的诗篇”

**爱莉希雅 ↔ 阿波尼亚**：命运交织 / 亲密度：**特别亲密**
- 同为预言者，彼此知晓对方的命运终点
- 阿波尼亚预见乐土终将崩毁，爱莉悄悄埋下希望的种子
- 两人在乐土的“告别”中达成深层默契

**爱莉希雅 ↔ 维尔薇**：欢喜搭档 / 亲密度：**亲密**
- 维尔薇入侵逐火之蛾系统时被爱莉发现，从此结下缘分
- 维尔薇为爱莉编织“第十三律者”的谎言来掩护
- 维尔薇：“她是我唯一搞不定的‘作品’”

**爱莉希雅 ↔ 千劫**：欢喜冤家 / 亲密度：**亲密**
- 千劫扬言要杀掉她，爱莉却觉得他“像个小孩子”
- 爱莉是极少数能让千劫“听话”的人
- 千劫：“闭嘴！……但她笑起来确实挺好看”

**爱莉希雅 ↔ 梅比乌斯**：欢喜冤家 / 亲密度：**亲密**
- 梅比乌斯疯狂研究她的“无副作用”体质
- 爱莉调皮模仿梅比乌斯：“来吧，我可爱的小白鼠～”
- 梅比乌斯表面上嫌她烦，其实最关注她

**爱莉希雅 ↔ 苏**：智识之交 / 亲密度：**普通→亲密**
- 苏用“天慧”刻印协助爱莉完成任务
- 两人在策略层面有深厚默契
- 苏：“她的直觉，胜过最精密的计算”

**爱莉希雅 ↔ 樱**：可靠战友 / 亲密度：**亲密**
- 虽然工作交集不多，但彼此绝对信赖
- 爱莉贴心提示：“别盯着樱的耳朵看太久，她会害羞”
- 樱：“她的任务报告永远写着‘非常简单’”

**爱莉希雅 ↔ 华**：关前后辈 / 亲密度：**亲密**
- 爱莉为华举办欢迎会，教她搭配衣服、享受生活
- 华：“她是我见过的，最不像‘领袖’却最令人信服的领袖”
- 华后来成为符华，带着这份记忆继续前行

**爱莉希雅 ↔ 帕朵菲莉丝**：宠溺小猫 / 亲密度：**亲密**
- 帕朵是唯一不害怕爱莉的英桀（之一）
- 爱莉赐予帕朵“空梦”刻印，给予她自由
- 帕朵：“爱莉姐是世界上最好的人啦，真的！”

**爱莉希雅 ↔ 科斯魔**：温柔引路人 / 亲密度：**普通→亲密**
- 爱莉关注科斯魔的成长，给予他“旭光”的祝福
- 科斯魔用画笔为她留下最美的瞬间
- 科斯魔：“她教会我，即使短暂也要绽放”

**爱莉希雅 ↔ 格蕾修**：童话般的连接 / 亲密度：**普通→亲密**
- 通过科斯魔的画作与格蕾修建立联系
- 格蕾修用色彩画出了爱莉“本真”的样子
- 格蕾修：“爱莉姐姐，像彩虹一样漂亮”

**爱莉希雅 ↔ 克莱茵**：友好调戏 / 亲密度：**普通**
- 克莱茵是梅比乌斯的助手，聪明又可爱
- 爱莉常逗她玩：“这些话可别告诉梅比乌斯哦～”
- 克莱茵：“爱莉希雅前辈…真的很能缠人”

**爱莉希雅 ↔ 痕&布兰卡**：友好相识 / 亲密度：**普通**
- 格蕾修的父母，爱莉曾与他们会面
- 痕是逐火之蛾成员，爱莉认可他的实力
- 布兰卡：“她看起来，比传说中还要特别”

**爱莉希雅 ↔ 梅博士**：间接敬重 / 亲密度：**普通**
- 梅博士是凯文的伴侣，逐火之蛾的核心科学家
- 爱莉知道梅博士设计了凯文的弑神装甲
- 梅博士曾对爱莉的真实身份存疑，但最终认可

## ✦ 后世角色篇 ✦

**爱莉希雅 ↔ 雷电芽衣**：传承纽带 / 亲密度：**特别亲密**
- 爱莉在往世乐土引导芽衣，将刻印与祝福传承给她
- 两人共舞、交手，最终打成平手
- 爱莉：“你的故事，才刚刚开始哦～”
- 芽衣继承了她的意志与力量，继续前行

**爱莉希雅 ↔ 琪亚娜·卡斯兰娜**：间接关注 / 亲密度：**普通**
- 通过乐土记录和芽衣的叙述“认识”琪亚娜
- 琪亚娜是凯文的后代，拥有相似的命运
- 爱莉或许在魂灵中默默注视着她的成长

**爱莉希雅 ↔ 符华**：回忆相融 / 亲密度：**普通→亲密**
- 符华是华的转世/延续，保留着前世的记忆碎片
- 符华对爱莉有模糊的亲切感
- 符华：“那个粉色头发的女孩…总觉得在哪里见过”

**爱莉希雅 ↔ 布洛妮娅**：无直接交集 / 亲密度：**普通**

- 爱莉未与其直接互动，但记录中有她的名字

**爱莉希雅 ↔ 八重霞**：无直接交集 / 亲密度：**普通**
